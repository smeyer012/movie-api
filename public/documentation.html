MyFlix Documentation

A movie application that utilizes Node.js, Express, Mongo DBB, and Rest architecture.

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it's featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., “Thriller”)
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username, password, email, date of birth)
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites

URL Endpoints

Request URL HTTP Method Response
Get list of all movies /movies GET A JSON object holding data about all movies
Get information about a movie by title /movies/[Title] GET A JSON object holding data about a movie
Get information about a genre /genres/[genre] GET A JSON object holding data about a genre
Get information about a director /directors/[director] GET A JSON object holding data about a director
Get list of all users /users GET A JSON object holding data about all users
Get a user's data by ID or username /users/[ID or username] GET A JSON object holding data about a movie
Add a user by username /users/[username] POST A message indicating that addition of user was successful
Deletes a user by ID or username /users/[ID or username] DELETE A message indicating that removal of user was successful
Add or update a user's information by ID or username (add or update username, password, email, date of birth, etc) ** Note: a user's ID cannot be changed /users/[ID or username]/:datatype/:data PUT A message indicating that update of user's information was successful
Removes a user's information by ID or username (remove password, email, date of birth, etc) ** Note: a user's ID or username cannot be removed /users/[ID or username]/:datatype/:data PUT A message indicating that update of user's information was successful
Add a movie to user's list of favorites by ID or username /users/[ID or username]/favorites/[Movie] PUT A message indicating that addition of movie to user's favorites was successful
Remove a movie from user's list of favorites by ID or username /users/[ID or username]/favorites/[Movie] DELETE A message indicating that removal of movie from user's favorites was successful
